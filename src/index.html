Chrome-based browsers (Brave, Chrome, Edge) use App-Bound Encryption which prevents yt-dlp from reading cookies directly. Use Sign in with YouTube above, or export cookies manually:

  2. Log into youtube.com
  3. Click the extension icon → Export cookies for this site
  4. Save the file and select it in the cookies.txt field above
