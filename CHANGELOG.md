# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-04-05

### Added
- Download videos from YouTube, Instagram, Facebook, X, TikTok, and 1000+ sites via yt-dlp
- M3U8 / HLS stream capture via ffmpeg
- In-app authentication for YouTube, Instagram, and Facebook using WebView2 CDP cookie extraction
- YouTube playlist browser with per-video and bulk download
- File converter supporting MP4, MKV, WebM, MOV, AVI, GIF, MP3, M4A, AAC, FLAC, WAV, Opus
- Quality selection from 360p to best available, plus audio-only modes
- Real-time download progress and log output
- Persistent auth — cookies saved locally and restored on next launch
- Windows, macOS, and Linux support
